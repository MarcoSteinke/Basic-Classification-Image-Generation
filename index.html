Classification image generator

Instructions:

(1) Draw an input which includes the important features to classify an image using convolutional neural networks.

(2) Next step is to insert a label using the following input.

(3) Use the "save"-button to save the input as image file.

(4) Use the "generate"-button to generate another image by learning from the previous ones with the very same label.


In the future there will be a feature to download a ZIP-file containing a chosen amount of generated inputs with a certain label.
